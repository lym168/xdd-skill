## 核心概念

### 认知体系
- **认知操作系统** — 由天赋、原生家庭、后天信息构成的思维框架
- **1.01法则** — 每天进步1%的复利效应
- **简单相信** — 抛开先入为主观念直接采纳建议的能力
- **换位思考** — 理解对方的利益结构和决策逻辑

### 职业规划
- **个人即公司** — 将自己视为一家公司进行战略规划
- **选择大于努力** — 找工作时投入资源比工作时更重要
- **相对优势** — 田忌赛马式竞争策略
- **被看见** — 主动让关键人物注意到你的价值

### 原生家庭（萨提亚模式）
- **三维结构** — 人员+关系+认知
- **重要他人** — 弥补原生家庭功能缺失
- **权威模式** — 无权威/多权威/权威挑战/权威恐惧
- **六种应对模式** — 控制/打岔/讨好/拒绝/共生/替代

### 冥想与心力
- **高我 vs 小我** — 智慧自我与恐惧自我
- **勇敢即爱自己** — 直面恐惧的修行
- **工具人陷阱** — 忘记本心

将 `xdd-perspective/` 目录复制到 WorkBuddy 的 skills 目录下（`~/.workbuddy/skills/`），即可在对话中通过关键词触发许单单视角的数字分身。
