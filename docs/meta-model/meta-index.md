## 领域对象（核心概念）

### 认知体系（已确认事实）
- **认知操作系统** - 由天赋、原生家庭、后天信息构成的思维框架
- **简单相信** - 高级能力，指抛开先入为主观念直接采纳建议的能力
- **信息茧房** - 社会阶层固化由认知和信息圈层决定
- **负能量PUA** - 网络负面信息对个体的精神操控
- **1.01法则** - 每天进步1%的复利效应

### 原生家庭理论（已确认事实）
- **萨提亚模式** - 存在人本主义家庭治疗流派
- **三角关系** - "我-妈妈-爸爸"三方关系结构
- **原生家庭结构** - 人员结构、关系结构、主观认知结构
- **三代影响** - 爷爷奶奶→父母→我→孩子

### 职业规划方法论（已确认事实）
- **个人即公司** - 将个人视为一家公司进行战略规划
- **选择大于努力** - 找工作时投入资源比工作时更重要
- **能力 vs 机会** - 能力是基础，展示能力获得认可是关键
- **绩效显性/隐性** - 高学历应避免绩效显性工作（如销售）

### 冥想修行（已确认事实）
- **高我 vs 小我** - 智慧的自我 vs 恐惧驱动的自我
- **勇敢即爱自己** - 核心结论：人生最重要的是勇敢
- **工具人陷阱** - 忘记本心，把自我变成达成目标的工具
