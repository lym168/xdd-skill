| 05-原生家庭对个人发展的影响与亲密关系的探讨.md | 第5讲 | 父母关系四类型、自我完整性 | 已确认事实 |

**关键理论**:
- 存在人本主义流派（Virginia Satir, 1950s）
- 三代影响模型（爷爷奶奶→父母→我→孩子）
- 三角关系理论
- 四种关系线（亲密/一般/疏离/冲突）
- 人员-关系-认知三维结构
- 重要他人（非亲生父母的心理替代者）
- 妈妈的三大功能：亲（皮肤接触）、安全感、连接、归属
- 爸爸的四大影响：自信、内在力量、价值感、清晰自我形象
- 权威的四种模式：无权威、多权威、权威挑战、权威恐惧
- 自我完整性五主题（18岁前）：独立自主、个性独特、冒险挑战、竞争合作、成功挫败
- 父母关系四类型：亲密、冲突、疏离、一般
- 六种应对模式：控制/服从、打岔回避、讨好依赖、拒绝说教、共生妈宝、替代捆绑
- 一致性关系模式（健康模式）
- 改变原生家庭影响的六种方法：看书、学习、寻找重要他人、职业规划、改变模式、心理治疗
