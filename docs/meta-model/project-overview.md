# Project Overview

## 项目性质

这是一个**个人知识库/文档集合项目**，非传统代码项目。核心是许单单（拉勾网前CEO）关于个人成长、职业规划、原生家庭、冥想修行的文档化知识体系。

## 项目目标

将隐性知识显性化，通过：
- 课程转录（10讲认知训练营 + 6讲原生家庭课程）
- 深度访谈（凉子访谈、公开采访）
- 个人反思（40岁感悟、冥想心得）
- 商业思考（企业战略与个人规划）

形成可复用的认知操作系统方法论。

## 内容架构

### 四大主题模块

```
1. 职业规划与认知训练（career_design_meta_knowledge/）
   ├── 核心理论：认知OS、简单相信、信息茧房、负能量PUA
   ├── 方法论：机会vs能力、职业规划、长期主义
   ├── 实战案例：弟弟pony的1.01成长、内向者转型
   └── 高阶思维：贵人人脉、换位思考、绩效显性

2. 原生家庭与心理治疗（origin_family/ + Satir family therapy model/）
   ├── 个人成长史：农村出身、父亲早逝、考入北大
   ├── 理论体系：萨提亚模式家庭治疗
   ├── 结构分析：人员/关系/认知三维结构
   └── 疗愈实践：小组讨论、觉察训练

3. 冥想与元认知（meditation&&Metacognition/）
   ├── 核心修行：身心灵疗愈、勇敢面对
   ├── 哲学反思：放下执念、托举他人
   ├── 人生感悟：40岁总结、直面困难
   └── 认知升级：高我vs小我、工具人陷阱

4. 商业与职场哲学（commercial&&career/）
   ├── 企业战略：个人即公司
   ├── 竞争策略：守正出奇、资源整合
   └── 职场生存：35岁天花板、996选择
```

## 文档生产流程

**已确认事实** - 文档格式高度统一：

```
## 关键词
## 全文摘要（AI生成，200-300字）
## 章节速览（时间戳分段，00:00格式）
## 思维导图（阿里云OSS外链）
## 发言总结（发言人视角）
## 问答回顾 / 要点回顾（Q&A格式）
```

**推断** - 生产工具可能是：
- 录音 → AI转录 → 人工校对 → 标准化模板
- 或半自动化的内容管理系统

## 知识体系特征

### 核心循环
```
认知偏差 → 原生家庭卡点 → 情绪困扰 → 行为模式 → 结果不理想 → 强化认知偏差
```

### 破局路径
```
觉察（冥想） → 理解（原生家庭分析） → 重构（认知OS升级） → 行动（勇敢实践）
```

## 目标受众

**高概率推断**：
- 一线城市职场人（25-40岁）
- 有原生家庭创伤或卡点
- 对自我成长有强烈需求
- 有一定经济能力（课程收费较高）
- 互联网/金融行业从业者为主

## 项目规模

- **文档数量**: 26个Markdown文件
- **总容量**: 约2.5-3万字（未精确统计）
- **时间跨度**: 2021年-2025年（基于文档日期）
- **课程规模**: 认知训练营10讲 + 原生家庭课程6讲 + 冥想课程多期

## 技术特征

- 纯Markdown格式，无代码
- 依赖外部图片服务（阿里云OSS）
- 目录命名含特殊字符`&&`
- 无构建/测试/部署流程
- 只读知识库性质
