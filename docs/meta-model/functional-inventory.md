## 内容生产流程（反向工程）

**高概率推断**:

```
录音/访谈 → AI转录（带时间戳） → 人工校对 → 关键词提取 → 全文摘要生成 → 章节速览整理 → 思维导图制作（Mermaid/ProcessOn） → 上传OSS → 最终Markdown输出
```

**证据**:
1. 所有文档带精确时间戳（格式：`##### 00:00:00 章节标题`）
2. 存在"全文摘要"和"章节速览"两个层次
3. 思维导图统一为外链图片（阿里云OSS）
4. 文件名高度格式化（`X.0-主题_导读/原文by许单单X.md`）
